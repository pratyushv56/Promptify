Privacy Policy for Promptify

Last updated: March 2026

Overview

Promptify is a browser extension designed to help users improve prompts using artificial intelligence. This privacy policy explains how information is processed when using the extension.

Information Processed

When a user chooses to use Promptify, the prompt text entered by the user may be sent to our backend server to generate an improved version of the prompt.

Data Usage

The prompt text is used solely to generate an improved prompt and return it to the extension. Promptify does not use this information for advertising, profiling, analytics, or tracking users.

Data Storage

Promptify does not store user prompts or personal information in a database. Prompt text is processed by the backend service only for generating the improved prompt and is not retained after processing.

Data Sharing

Prompt text may be processed by third-party AI service providers used by our backend infrastructure solely for the purpose of generating the improved prompt. Promptify does not sell, trade, or share user data for advertising or marketing purposes.

Tracking

Promptify does not collect browsing history, personal identifiers, or other information for tracking users across websites. No background data collection occurs.

User Control

Promptify only processes text that users explicitly choose to submit for prompt improvement. The extension does not automatically collect or process webpage content.
